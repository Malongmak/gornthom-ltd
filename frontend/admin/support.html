Common Issues

Backend not connecting expand_more

Make sure you ran npm start inside the backend/ folder. The server must be running on port 3000. Check Settings to verify the API URL.

Paystack payment fails expand_more

Ensure PAYSTACK_SECRET_KEY is set in backend/.env. The public key in packages.html must match your Paystack dashboard.

MikroTik not activating connections expand_more

Set MIKROTIK_HOST, MIKROTIK_USER, and MIKROTIK_PASSWORD in backend/.env. The backend server must be on the same network as the router. API port 8728 must be open on the router.

Sessions lost after server restart expand_more

Sessions are stored in memory. For persistence across restarts, a database (SQLite or PostgreSQL) needs to be added to the backend. This is a known limitation of the current setup.
